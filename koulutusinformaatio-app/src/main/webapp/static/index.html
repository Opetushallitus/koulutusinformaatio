Opintopolku on kehittyvä palvelu

Opintopolusta löydät nyt syksyn ammatillisen koulutuksen yhteishaussa mukana olevat koulutukset. Voit etsiä hakukoneella kiinnostavia koulutuksia ja hakea niihin. Kannattaa perehtyä myös hakuohjeisiin ja valintaperusteisiin.

Opintopolku kehittyy 2013–2014

Luvassa on helppokäyttöinen palveluportaali, josta jatkossa löydät kaiken tutkintoon johtavan koulutuksen ja tietoa muista opiskelumahdollisuuksista. Voit tutustua vaihtoehtoihin ja hakea koulutukseen Opintopolun kautta.

Uutta lokakuussa 2013

  • Syksyllä 2014 alkava ammatillinen koulutus ja lukiokoulutus
  • Tietoa ja tarinoita opiskelumahdollisuuksista

Uutta joulukuussa 2013

  • Muu peruskoulun jälkeinen koulutus, esimerkiksi kymppiluokat ja ammattistartit

Uutta helmikuussa 2014

  • Ammatillisen koulutuksen ja lukiokoulutuksen yhteishaku

Uutta kesäkuussa 2014

  • Korkeakoulujen koulutus, johon haetaan syksyllä 2014
  • Haku muuhun peruskoulun jälkeiseen koulutukseen

Uutta syyskuussa 2014

  • Korkeakouluhaku
  • Aikuiskoulutus Opintopolussa
